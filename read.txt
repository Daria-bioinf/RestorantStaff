TREE	Показать всю структуру классов
CD <узел>	Перейти в нужный узел дерева (Cleaner, ChiefCooker, PastyCook, Sushef, Hostess)
MO <имя>	Создать новый объект в текущем листе
DO <имя>	Удалить объект по имени
MDO <имя>	Модифицировать объект по имени
DIR	Показать список всех объектов в текущем листе или поддеревьях
SHOW <имя>	Показать полную информацию об объекте
SAVE	Сохранить все объекты в файл staff.txt
READ	Загрузить все объекты из файла staff.txt
EXIT	Завершить работу программы
